Academic Projects at University of Michigan

ROB 550: Robotic Systems Lab

Mobile Inverted Pendulum System

This project involved the design and development of a two wheeled Mobile Inverted Pendulum system. Using a cascaded control architecture the bot was balanced, and driven manually via a DSM remote and autonomously in pre-defined trajectories.

NA 568: Mobile Robotics

Invariant Extended Kalman Filtering for Robot Localization using IMU and GPS

This project aims to implement an In-EKF based localization system and compare it against an Extended Kalman Filter based localization system and a GPS-alone dataset. We will use the UM North Campus Long-Term Vision and LIDAR dataset, an autonomy dataset for robotics research collected on the University of Michigan North Campus. It consists of data from several sensors including planar lidar, omnidirectional camera, IMU, and GPS.

EECA 504: Foundations of Computer Vision

Object Tracking for Safety

The aim of this project was to use object tracking to detect and draw conclusions about the distance of the moving object from the camera, and additionally issue a warning based on the proximity of the object to the camera. Such a warning would be useful for a robot, vehicle, or even human, to react appropriately when the camera is mounted on them. This is implemented with a depth estimator method that makes use of the variation in scale of the bounding box with depth of the object.
