Academic Projects at University of Michigan

ROB 550: Robotic Systems Lab

Mobile Inverted Pendulum System

This project involved the design and development of a two wheeled Mobile Inverted Pendulum system. Using a cascaded control architecture the bot was balanced, and driven manually via a DSM remote and autonomously in pre-defined trajectories.

  • Open in GitHub

6-DOF Serial Link Robotic Manipulator

As the title suggests, this project involved designing a gripper for a manipulator and writing the codebase in Python to drive the serially connected motors autonomously; in conjuction with using kinect camera suite (3D depth sensor and RGB camera).

  • Open in GitHub

Slam and Path Planning implementation on MBot

This project was supposed to be implemented on MBot, a bot developed in-house at UoM for learning purposes, but was rather completed virtually on a simulated PyGame environment due to the COVID-19 pandemic.
